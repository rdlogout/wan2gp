# Wan2GP - Advanced Video Generation

This is a powerful video generation application that supports multiple AI models for both text-to-video and image-to-video generation. It features advanced controls, LoRA support, and various preprocessing options.

## Features

- Multiple video generation models (WAN, LTX Video, Hunyuan Video)
- Text-to-video and image-to-video generation
- LoRA (Low-Rank Adaptation) support
- Advanced prompt engineering
- Video preprocessing and masking
- Queue-based batch processing

## Usage

Simply run the application and use the web interface to:

1. Select your preferred model
2. Enter your text prompt or upload images
3. Adjust generation parameters
4. Generate high-quality videos

Check out the configuration reference at https://huggingface.co/docs/hub/spaces-config-reference
